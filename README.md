# Quick Restart
Mod for quickly reloading rooms in Slay the Spire 2 for the classic save scumming experience. Supports both english and simplified chinese.

## Usage
When opening the pause screen (escape) there will be a new button "Restart Room" that will reload the current rooma dn undo everything since the last auto save.

This can be used to undo event choices or combats as long as they're not completed already.

## Installation
Make a folder called `mods` in the same directory as your sts2.exe `Steam\steamapps\common\Slay the Spire 2` and unzip the contents of the [release (the zip)](https://github.com/erasels/StS2-Quick-Restart/releases/tag/v1.0) into that folder or a subdirectory within that folder.
