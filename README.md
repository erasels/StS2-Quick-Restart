# Quick Restart
Mod for quickly reloading rooms in Slay the Spire 2 for the classic save scumming experience.

## Usage
When opening the pause screen (escape) there will be a new button "Restart Room" that will reload the current rooma dn undo everything since the last auto save.

This can be used to undo event choices or combats as long as they're not completed already.

## Known Issues
The button won't do anything if it's used during multiplayer but still shows and is clickable.

## Installation
Make a folder called `mods` in the same directory as your sts2.exe `Steam\steamapps\common\Slay the Spire 2` and unzip the contents of the release (once available) into that folder or a subdirectory within that folder.
